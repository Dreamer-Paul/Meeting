3. 有奖问答 - 温故而知新

1. 在 Linux 课程里，查看路由跳数的命令是？

在 Linux 里查看路由跳数使用的命令是 traceroute

2. 在 JavaScript 课程里，编写代码应在什么标签之间？

编写 JavaScript 代码应在 Script 标签之间

3. 在思科课程里，修改主机名的命令是？

在思科设备里，修改主机名的命令是 hostname

4. 在 JavaScript 课程里，哪个单词指的字符串对象？

String 意为 JavaScript 中的字符串对象

5. 在 Linux 课程里，touch /home/paul/interesting.txt 命令有什么作用？

在 /home/paul/ 目录下新建一个名为 “interesting.txt” 的文件

6. 在思科课程里，interface (?) FastEthernet0/1 - 6 命令内的答案应是什么？

答案为 range。使用 interface range FastEthernet0/1 - 6 命令可以同时设置1到6的高速端口

7. 在 Linux 课程里，ping -c 10 www.baidu.com 命令有什么作用？

发十个 ping 包给 www.baidu.com

8. 在 JavaScript 课程里，下面的哪个声明属于私有属性？

使用 var 创建的为私有属性（属性前后有下划线），使用 this 创建的为公有属性

9. 在思科课程里，有哪几个生成树协议是思科独有协议？（多选）

有 PVST+、快速 PVST+ 和 MSTP 共三种思科独有协议
